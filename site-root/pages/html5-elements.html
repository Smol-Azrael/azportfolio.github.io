Header

• The header tag is used to defines the header of a document or specific section. This tag uses at the beginning of the contents of the collection of the navigation links.

Example

<body>
    <header>
        <h1>This works fine</h1>
        <p>Research is going well so far.</p>
    </header>
</body>

Nav

• The nav tag uses to define the collection of the links to the current page, or the other part of the page. Usually used to go with menu, index, or table of context. Also, the screen reader uses a nav tag to decide the to skip or not on the content at the first render for the people who cannot read.

Example

<nav> 
    <a href="../index">Index<a>
    <a href="../page/cascades.html">Cascades<a>
    <a href="../page/history.html">History<a>
</nav>

Aside

• The aside tag can be used to define the contents that connect to the main article but can be separated from it. It often used in the article that sits on the side.

Example

<p>Some new article about popsong.</p>

<aside>
<h4>Pop song</h4>
<p>The general information of the popsong.</p>
</aside>

Article

• The article tag uses to define the article that can be separated completely alone from other articles.

Example

<article>
<h2>Index</h2>
<p>The beginning page that simiply shows the basic concept.</p>
</article>

<article>
<h2>Page</h2>
<p>The collection fo the basic HTML page.</p>
</article>

<article>
<h2>CSS</h2>
<p>The style sheet that makes HTML page clean and nice.</p>
</article>

Section

• The section tag uses to define the independent article on the HTML document.

Example

<section>
<h2>Low-poly Object</h2>
<p>Low-poly Object refers to polygon mesh in modeling with small amount of polygons.</p>
</section>

<section>
<h2>High-poly Object</h2>
<p>High-poly Object refers to polygon mesh in modeling with great amount of polygon.</p>
</section>

Figure

• The figure tag uses to specifies the independent contents like the diagram, illustrate, or image. The figure does have a connection with the main article, but the figure content is usually isolated from the articles and the position should not affect the main article's readability.

Figcaption

• The figcaption used to describe and help the viewers to understand what is the figure item is telling to the viewer.

Example

<figure>
<img scr="../photo/06843_jet.jpg" alt="Double jets flying">
<figcaption>Double jets cross flying scene</figcaption>
</figure>

Footer

• The footer tag uses to identify information like author, copyright, contact, sitemap, and etc.

Example

<footer>
<p>Creator : AzraelModeling 2022.</p>
<p>Insta : @azraelmodeling</p>
<p><a href="mailto:azraelmodeling@gmail.com">Email to creator.</a></p>
</footer>
